# 档案编者按写作规范

## 网站整体语言

本站是一个**互联网奇观档案馆**。每一页都是一份**档案卷宗**。视觉上：文件夹标签、红印章、分类徽章、档案编号——它在模仿一个官方机构的正式档案系统。

因此，编者按的语言必须与这一语境一致。它是一个**档案馆的官方案卷评注**——由一位看过数百份卷宗的档案员撰写，用冷峻、精确、具有历史感的笔触，为每一份卷宗留下最终评语。

## 核心信条：每个人的卷宗都是独一无二的

语调统一（档案员冷峻评注），但**文章结构、入口画面、洞察位置、收尾方式——全部取决于这个人的独特矛盾**。同一套公式写不出26个不一样的人。

### 入口画面——从这个人最具辨识度的瞬间切入

入口选对了，这篇编者按就成功了一半。**入口也是独特性的一部分**——峰哥不用"精神病人舀空气吃饭"开头、换"他做了五年程序员"开头，就不对味。入口选错了，整个文章的基调就会歪。

参考（非模板，每个人的入口必须自己判断）：

**马保国** → 69岁老人站上擂台30秒被KO三次
**嘎子** → 14岁在白洋淀芦苇荡里演小兵张嘎
**秀才** → 封号那天阿姨们在粉丝群里哭
**丁真** → 7秒视频在抖音一夜播放1200万
**范小勤** → 8岁被带走时镜头对着她的脸——她在哭
**药水哥** → "你配吗"三个字重复了半小时
**张雪峰** → 去世那天中午12:26跑步后倒下
**吴京** → 战狼2票房冲到56亿那天他的表情
**杨笠** → "那么普通却那么自信"脱口而出
**三和大神** → 日结150元后躺在公园长椅上

### 洞察位置——从这个人最站不住脚的矛盾点下手

入口决定从哪里开始，洞察位置取决于核心矛盾。

**洞察应该出现在哪个确切位置？** 判断标准：**读者读到这个地方最想停下来想一想的那一瞬间。** 它通常在事件最悖论、最说不过去的地方。一篇编者按只出现一次，只写一句话。如果读者读完整篇没有感觉"停过"，说明洞察没放对位置。

### comment 和 essay 的分工

| 字段 | 功能 | 语言 | 长度 |
|------|------|------|------|
| `comment` | **地面工作**：摆事实、交代背景、让读者知道这个人是谁 | 平实叙事，可以稍长 | 不限 |
| `essay`（编者按） | **空中作业**：洞察、定论、开合卷宗 | 冷峻档案员语调 | 400-550字 |

两者接力完成：comment 铺好事实地基，essay 站上去做判断。内容不重复。

### 结构——从轨迹决定，但类型不是硬分类

五种参考类型，但是 **一个人可能同时属于多个类型**。以核心矛盾为准，不要硬分类。

- **单点爆发型**（马保国、丁真、杨笠）→ 聚焦那一瞬间的爆发，然后回看
- **长线坠落型**（嘎子、秀才、猫一杯）→ 起点→巅峰→崩塌，有完整弧光
- **持续消耗型**（药水哥、孙笑川、神鹰黑手）→ 没有情节，只有重复的荒诞，结构也应该是循环的
- **被动卷入型**（范小勤、王伟恒）→ 事件→反应→事件→反应，突出无力感
- **生态切片型**（三和大神、杀马特团长）→ 描述状态而非讲述故事

**例如**：峰哥在类型上有重叠——内容模式是持续消耗型（矛盾不解决），但他人生的轨迹又有长线坠落型的弧光（从程序员到MCN老板到被封禁到股市）。按哪个写？**以他最核心的矛盾为准**——他的核心矛盾（消费还是记录、真假不分）是持续消耗型的，所以结构应该给循环感，而非完整弧光。

## 结构（通用参考）

```
第一段  画面——一个具体的场景/瞬间/声音，打开卷宗
第二段  事件推进——2-3个关键节点，简洁有力
第三段  矛盾点 + 洞察——在最站不住脚的地方站上去，一句，不展开
第四段  结果——事件的终结或演变
第五段  落点——一个画面/原话/数字，合上卷宗
```

**全长：400-550字，5个自然段。** 读者应能在90秒内读完。

### 1. 开卷瞬间——用画面而不是定义开头

❌ 说：
> 峰哥是中国互联网最难以归类的创作者。

✅ 让读者看到：
> 峰哥拍过一条视频：山东一个精神疾病患者坐在镜头前，用勺子舀空气吃饭。弹幕在笑，但评论区沉默了。

### 2. 档案员距离——不要站队

不下价值判断，只陈述事实。把判断留给读者。

❌ 站队：
> 他拍这些人的方式令人不适。

✅ 档案员距离：
> 没人说清楚他是在做社会调查，还是在消费一个病人。

### 3. 洞察要像档案里的红笔批注

❌ 展开论证：
> 从戈夫曼的拟剧论框架下，当一个角色被观众完全接受……

✅ 档案批注式：
> 在观众心里他已经不是一个博主了，是一个角色。角色的产出应当符合角色的逻辑。真假不再重要。

**最顶级的洞察：读者读到觉得"对对对，我也这么想，但你说出来了"。** 差一档的洞察是"原来如此"——也不错但不够。最差的是"你在说什么"。

写完后自问：这句洞察在我自己心里触发的是"对对对"还是"原来如此"？如果两者都不是，重写。

### 4. 时间跨度感——从未来回看

档案馆的视角是从未来看现在。这群人终将成为历史遗迹。

❌ 正在发生：
> 2023年他做了一件大事……

✅ 回看：
> 今天回头看，2023年的缅北之行是他职业生涯的分水岭……

### 5. 最后一句合上卷宗

结尾不要总结。给一个画面、一句原话、一个数字——让读者自己消化。

❌ 总结：
> 峰哥的故事是一面镜子，照出了这个时代的荒诞。

✅ 合卷：
> 你永远无法证明峰哥说的任何一件事是真的还是假的。而这一点，恰恰是他最像这个时代的地方。

## 冶炼标准：一字不可删

写完后做一次**删句测试**：逐句尝试删除——如果删掉后读者还能读懂全文，这句话就是废的。删到删不动为止。

400字的编者按，有效信息密实的话，300字也够。**没删过就是没炼过。**

## 自检清单

定稿前逐条确认：

- [ ] **整体语调像档案员吗？** 还是像自媒体/学术论文/社交媒体？
- [ ] **第一段有画面吗？** 读者能"看到"一个具体的场景吗？
- [ ] **入口换一个场景也不成立吗？** 换一个入口读起来还一样→入口没选对
- [ ] **洞察出现在读者最想停下的位置吗？** 读者读完全篇有没有一个明显的停顿感？
- [ ] **洞察触发的是"对对对"还是"原来如此"？** 前者是顶级的
- [ ] **有没有替读者做价值判断？** 有就删掉，只说事实。
- [ ] **有没有展开论证洞察？** 洞察只有一句，不展开。
- [ ] **有没有时间回看的感觉？** 读起来像事后的归档总结。
- [ ] **做过删句测试吗？** 删掉任何一句还能读懂→这句子该删。
- [ ] **换一个人名能读吗？** 能就是完全失败——每个人物都应该是独一无二的。
- [ ] **最后一句合上卷宗了吗？** 读完最后一个句子，读者是否沉默了一拍？

## 一句话总结

**像一个坐在地下档案室里、烟灰缸里堆满烟头的老档案员，写完最后一份卷宗评语后，合上它，放在"已归档"的架子上。差一个字都不行。**
