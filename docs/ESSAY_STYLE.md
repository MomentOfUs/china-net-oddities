# 档案编者按写作规范

## 网站整体语言

本站是一个**互联网奇观档案馆**。每一页都是一份**档案卷宗**。视觉上：文件夹标签、红印章、分类徽章、档案编号——它在模仿一个官方机构的正式档案系统。

因此，编者按的语言必须与这一语境一致。它不是博客文章，不是新闻报道，不是B站评论区。它是一个**档案馆的官方案卷评注**——由一位看过数百份卷宗的档案员撰写，用冷峻、精确、具有历史感的笔触，为每一份卷宗留下最终评语。

## 语气定位

### 核心语调：冷峻的档案员

想象一位老档案员坐在地下档案室里，桌上摊着这份卷宗。他抽着烟，翻了翻前面的材料，然后在最后一页写下评语。他的语气：

- **冷静** 不是冷漠，而是不需要激动——他见过太多份卷宗了。
- **精确** 每句话都有事实和具体细节支撑，不空谈。
- **历史感** 他知道这些人终将被遗忘，但他还是要把他们记下来。
- **偶尔锋利** 在事件的矛盾处，他会停下来，写一句击中要害的判断，然后继续写下一份。

### 参考语感

> "一份互联网奇观人物的案卷评注。编者已审阅全部材料。以下为结案意见。"

这种语感。不是和你聊天的语感，是**出具报告**的语感。

### 绝对避免的语感

- ❌ 自媒体口吻（"让我们来看看…"、"不得不说…"）
- ❌ 社交媒体口吻（"家人们谁懂啊"、"谁还记得…"）
- ❌ 学术论文口吻（"从XX理论视角分析…"）
- ❌ 打Call口吻（"他真的我哭死"、"太牛了"）
- ❌ 道德审判口吻（"这是不对的"、"应该反思"）

## 核心信条：每个人的卷宗都是独一无二的

这是整份规范中最重要的一条。语调统一（档案员冷峻评注），但**文章结构、入口画面、洞察位置、收尾方式——全部取决于这个人的独特矛盾**。同一套公式写不出26个不一样的人。

### 入口画面——从这个人最具辨识度的瞬间切入

不同的人有不同的入口。入口选对了，这篇编者按就成功了一半：

**马保国** → 69岁老人站上擂台30秒被KO三次。他的整个传奇从这里开始，之前的事不重要。
**嘎子** → 14岁在白洋淀芦苇荡里演小兵张嘎。这是他一生最高光的时刻，此后全是下坡路——落差就是全部叙事。
**秀才** → 封号那天阿姨们在粉丝群里哭。他不为人知的另一面不是内容，是受众。
**丁真** → 7秒视频在抖音一夜播放1200万。他的全部就是这7秒，写多了多余。
**孙笑川** → 弹幕刷"孙狗"，他对着屏幕骂了三个小时。他的本质是被消费的符号，不是创作者。
**范小勤** → 8岁被带走时镜头对着她的脸——她在哭。她全程被动，从被选中到被送回，自己没有任何选择。
**药水哥** → "你配吗"三个字重复了半小时。他的艺术就是消耗自己，没有情节才是情节。
**张雪峰** → 去世那天中午12:26跑步后倒下。一个一辈子在跑的人，最终停在了跑道上。
**吴京** → 战狼2票房冲到56亿那天他的表情。他不是一个人，他是中国民族主义的票房计量单位。
**杨笠** → "那么普通却那么自信"脱口而出。五个字定义了她，也定义了一场社会战争。
**东北雨姐** → 穿着红色棉袄在田里喊"穿上！！"。她的所有标签都在这两个字里——粗粝、野蛮、真实、然后被证伪。
**三和大神** → 日结150元后躺在公园长椅上。不是一个人，是一种生存状态。

### 洞察位置——从这个人最站不住脚的矛盾点下手

入口决定从哪里开始，洞察的位置取决于核心矛盾在哪里：

**峰哥** → 矛盾：在拉近距离和消费距离之间无法自证。洞察位置：拍到精神病人之后。入笔：「没人说清楚是在做社会调查还是在消费一个病人」
**猫一杯** → 矛盾：好故事比真故事跑得快，但真相反噬。洞察位置：警方通报之后。入笔：「愤怒的不是被骗，是这个好故事被戳穿了」
**嘎子** → 矛盾：拼命想证明不只是嘎子，最终连嘎子也保不住。洞察位置：潘嘎之交之后。入笔：「他活成了自己扮演的角色的反面」
**丁真** → 矛盾：一个人同时站在公正和福祉的两端，无法两全。洞察位置：十年寒窗争议之后。入笔：「他什么都做不了，只是站在了那道算术题的中间」
**马保国** → 矛盾：最认真的打脸制造了最大的快乐。洞察位置：被KO后拍视频辩解。入笔：「他的大意了没有闪不是辩解，是实验艺术的起点」
**张雪峰** → 矛盾：一个怕穷的人拼到了累死。洞察位置：去世后旧视频被翻出。入笔：「穷过的人知道每分钱都是命换的」
**范小勤** → 矛盾：没有选择权却被整个互联网围观。洞察位置：被送回村子后。入笔：「她是这场荒诞剧里唯一没有台词的角色」
**药水哥** → 矛盾：到底是真的疯了还是在演疯了。洞察位置：你配吗循环之后。入笔：「他笑着反问的那一刻，你也不知道答案」
**王伟恒** → 矛盾：跑到自由的土地上失去了一切自由。洞察位置：街头举着HUNGRY牌子。入笔：「他走线越过的不是国境线，是中产幻觉的临界点」

### 结构——从轨迹决定

不要套五段公式。结构要反映人物的人生轨迹：

- **单点爆发型**（马保国、丁真、杨笠）→ 聚焦那一瞬间的爆发，然后回看
- **长线坠落型**（嘎子、秀才、猫一杯）→ 起点→巅峰→崩塌，有完整弧光
- **持续消耗型**（药水哥、孙笑川、神鹰黑手）→ 没有情节，只有重复的荒诞，结构也应该是循环的
- **被动卷入型**（范小勤、王伟恒）→ 事件→反应→事件→反应，突出无力感
- **生态切片型**（三和大神、杀马特团长）→ 描述状态而非讲述故事

### 判断标准

> **把这篇编者按里的人名换成另一个人，还能读——说明你没有理解这个人的独特性。**

写完后自问：
- 入口画面换到另一个人身上合适吗？→ 合适→重新选
- 洞察在另一个人的矛盾上也成立吗？→ 成立→洞察不够精准
- 结构拿到另一个类型也能用吗？→ 能用→结构太通用

档案员的语调是统一的。但26份卷宗永远不应该长得一样。

## 结构（通用参考）

### 1. 开卷瞬间——用画面而不是定义开头

不要定义一个人，让读者先"看到"他。

❌：
> 峰哥是中国互联网最难以归类的创作者。

✅：
> 峰哥拍过一条视频：山东一个精神疾病患者坐在镜头前，用勺子舀空气吃饭。弹幕在笑，但评论区沉默了。

第一条是在"说"，第二条是在"让读者看到"。

### 2. 档案员距离——不要站队

档案员不下价值判断。他陈述事实，把判断留给读者。

❌ 站队：
> 他拍这些人的方式令人不适，他是在消费苦难。

✅ 档案员距离：
> 没人说清楚他是在做社会调查，还是在消费一个病人。这个问题追了他五年。他从来不解释。

### 3. 洞察要像档案里的批注

想象档案页的空白处，档案员用红笔写了一句批注。只写一句。不展开。

❌ 展开论证：
> 这反映了在戈夫曼的拟剧论框架下，当一个角色被观众完全接受，真假就不再重要。观众需要这个角色持续产出符合其设定的内容……

✅ 档案批注式：
> 在观众心里，他已经不是一个博主了，是一个"亡命天涯"的角色。角色的产出应当符合角色的逻辑。真假不再重要，"好玩"才重要。

档案员不考试。他只要写对了，读者就知道对。

### 4. 时间跨度感——从未来回看

档案馆的视角是从未来看现在。这些人终将成为历史遗迹。这种时间距离感是档案语气的来源。

❌：
> 2023年他做了一件大事……

✅：
> 今天回头看，2023年的缅北之行是他职业生涯的分水岭……

❌：
> 他现在还在更新。

✅：
> 2025到2026年他还在更新。如果说有什么比"一给我里giao giao"更值得记住的，就是这个事实本身：互联网早就不记得他为什么出名了，只记得他还在。

### 5. 最后一句合上卷宗

结尾不要总结。要给一个画面、一句原话、一个数字——读者看完后会自己消化。

❌：
> 峰哥的故事是一面镜子，照出了这个时代的荒诞与矛盾。

✅：
> 你永远无法证明峰哥说的任何一件事是真的还是假的。而这一点，恰恰是他最像这个时代的地方。

❌：
> 阿giao的经历值得每一个人思考。

✅：
> 他推着小车等了十年等不到的机会，互联网用一个声音就给了他。现在他只是在拼命证明自己配得上这份运气。

## 完整示例

### 峰哥

> 峰哥拍过一条视频：山东一个精神疾病患者坐在镜头前，用勺子舀空气吃饭。弹幕在笑，但评论区沉默了。
>
> 没人说清楚他是在做社会调查还是在消费一个病人。这个问题追了他五年。他拍流浪汉、拍26岁癌症青年在出租屋里等死、拍台风吹垮的烂尾楼、拍缅甸诈骗园区里被人用枪顶着头的中国人。他的镜头永远对准社会最边缘的角落，而且从来不解释。你问他，他就回一句"这是好事儿啊"。
>
> 2023年下半年，他去了一趟缅甸。回来之后说自己在那边被持枪人员挟持，差点被枪毙，起飞回国时哭了十多分钟。有人查了缅甸签证时间线，说对不上。但更多人不在乎——因为在观众心里，他已经不是一个博主了，是一个亡命天涯的角色。角色的产出应当符合角色的逻辑。真不真不重要，好玩就值了。
>
> 2024年，全平台一起关了他的门。微博、B站、抖音、小红书全面禁止关注。没有明确原因。一个总是在拍边缘的人，最终自己也被赶到了边缘。
>
> 被封之后他转了股市，2026年拿着手机截图说赚了一个亿。"这是好事儿啊"——这句话从他嘴里说出来，永远有效。你永远无法证明峰哥说的任何一件事是真的还是假的。而这一点，恰恰是他最像这个时代的地方。

### 猫一杯

> 2024年4月12日，杭州警方发布了一则通报：千万粉博主徐某某与同事薛某共同策划了"秦朗巴黎丢作业"事件，网购作业本自拍造假。双方被依法行政处罚。
>
> 但比造假本身更有意思的是事情发生的过程。那条视频精准踩中了每一个传播学上的引爆点：异国元素、文化认同、情绪价值、完美闭环。在警方通报之前，全网沉浸在一次寻人运动的热潮里。秦朗是谁不重要——重要的是互联网已经生产了一个足够好的故事。好故事永远比真相跑得快。
>
> 然后故事崩溃了。北京警方、杭州警方、教育局——三方接力调查，最后给出结论：不存在。愤怒的观众不是气被骗了——是气这个故事是假的。如果它是真的，多完美的文化图景。但它是假的，所以所有人都成了傻瓜。消费的从来都不是真相，消费的永远是叙事。
>
> 猫一杯的账号被全平台永久封禁。她发了一段道歉视频，措辞诚恳但于事无补。
>
> 她留下的不是一个道歉。是一个教科书级的案例：注意力经济的底层逻辑不是内容为王，是故事为王。而当故事被证明是假的，写故事的人会被吃掉。这是每一个网红都懂的恐惧，也是每一个网红都假装不存在的终局。

### 丁真

> 2020年11月11日，一个摄影师拍到一个藏族少年来买泡面。随手发到抖音，一夜播放量超过1200万。七秒视频。这就是丁真走红所需的全部原材料。
>
> 所有人都在等他开口要价。千万合约、综艺、电影——每一样都在等他签。但他签了国企。月薪3500元。理塘文旅的总经理说了一句话："我们不希望丁真成为一件商品，我们希望他成为一个人。"
>
> 然后他承受了中国互联网男性群体最尖锐的一次群体愤怒。"十年寒窗不如丁真一笑"——一个小学三年级学历的放牛娃，凭什么比我火？凭什么上联合国？这股愤怒如此之大，以至于"丁真"从一个名字变成了一把尺子：任何被认为是德不配位的人，都会被冠上"XX丁真"的标签。他不是在挨骂，他是在替所有人承受一个焦虑的投射：在一个不确定的世界里，努力到底还有没有用。
>
> 但还有另一组数据值得记录：甘孜州旅游收入在他走红后增长近三倍，理塘游客从15万增长到420万。修路、建学校、修供水工程——因为一个七秒视频，一个高海拔贫困县完成了一次不可逆的经济跃迁。那个放牛娃什么都没做错，他只是站在了一道永远无法两全的算术题中间：一边是私人公正，一边是公共福祉。
>
> 2026年，他仍然在理塘，仍然骑着马，仍然领着国企的工资。谢霆锋问他为什么那么开心，他说："因为我也很自由。"

## 自检清单

定稿前逐条确认：

- [ ] **整体语感对吗？** 像一个档案员在写案卷评注，而不是自媒体/社交平台/学术论文
- [ ] **第一段有画面吗？** 读者能"看到"一个具体的场景吗？
- [ ] **第三段有洞察吗？** 事件推进到最矛盾的位置，是否站上去发出了一针见血的判断？
- [ ] **洞察是一句批注，不是一段论证吗？**
- [ ] **有没有替读者做价值判断？** 有就删掉，只说事实。
- [ ] **有没有时间回看的感觉？** 读起来像是一个事后回望的总结，而不是正在发生的现场报道。
- [ ] **换一个人名能读吗？** 能就是完全失败——每个人物都应该是独一无二的。
- [ ] **最后一句合上卷宗了吗？** 读完最后一个句子，读者是否沉默了一拍？

## 一句话总结

**像一个坐在地下档案室里、烟灰缸里堆满烟头的老档案员，写完最后一份卷宗评语后，合上它，放在"已归档"的架子上。**
