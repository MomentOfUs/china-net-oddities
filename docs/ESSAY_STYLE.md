# 档案编者按写作规范

## 一句话定位

> **不是写这个人做了什么，是写"这个人为什么值得你停下来想三秒钟"。**

编者按不是一个博客、一篇百科、一个评论。它是一个**极短的人物特写**——用一个具体的场景或细节，让读者在一分钟内理解"为什么这个人代表了这个时代的一条裂缝"。

## 核心原则

### 1. 始于一个具体细节，不要始于一个抽象判断

❌ 不要：
> 阿giao的荒诞在于他是一个一无所有的人，却改变了命运。

✅ 要：
> 2017年的某个下午，一个推着小车卖苹果的河南青年，对着手机喊了一句没有任何意义的尾音——"一给我里giao giao"。他不知道那条视频会改变他的人生，因为他已经花了十年什么都没改变。

**力量来自"具体"**——时间、地点、动作、数字、声音。一个画面胜过十句判断。

### 2. 用事实说话，每句话里面必须有事实

每写一句判断，立刻跟上事实支撑：
- "他火了" → "2017年，一给我里giao giao在全网的模仿量超过了一亿次"
- "他转型了" → "2023年，他把镜头上移，从搞怪变成了语录，开始讲'人一定要有梦想'"
- "他很矛盾" → "他是国家一级登山运动员，也是蹲在路边和流浪汉分烟的那个人"

**如果一句话不能加一个事实，这句话就是废话。**

### 3. 情感来自读者自己完成，不是你塞给他的

不要写"这让人很难过"、"观众感到愤怒"。把场面摆出来，读者自己会难过。

❌：
> 那个26岁患癌无亲人照顾的年轻人，令人感到无比惋惜。

✅：
> 他拍了"全B站最惨的B友，26岁患癌症无亲人，在出租屋静静等死"。评论区有两种声音：有人感谢他记录，有人说他消费苦难。他什么都没回应——下一期视频拍的是赚了一个亿。

**不判断。把判断留给读者。**

### 4. 理论是底色，不是标题

不要喊理论名字。把理论消化成观察：
- "戈夫曼的拟剧论" → "他有两个人生：镜头前的和镜头后的。前者是精心布置的前台，后者没人见过。"
- "鲍德里亚的拟像" → "真相是什么已经不重要了——重要的是互联网已经生产了一个足够好的故事。"

**判断标准：一段话去掉所有术语还能读懂，理论用对了；读不懂，生硬。** 最后的底线是不用任何术语也可以，完全不要理论。打动人的永远是人本身，不是框架。

### 5. 收在具体上，不在抽象上

❌ 空收尾：
> 这反映了互联网时代的注意力经济本质。

✅ 具体收尾：
> 他最后一条微博停在"人生真好玩，下辈子还来"。再也没有更新。

❌ 空收尾：
> 一个人的荒诞经历本身就是一个时代。

✅ 具体收尾：
> 那个白洋淀芦苇荡里拿着木头枪的少年，最终活成了自己扮演的角色的反面。

**一个画面、一句原话、一个数字——收在这里，让读者自己消化。**

## 结构建议（不是公式）

不是必须按这个顺序，但一个正常的编者按应该覆盖这几个层次：

```
第一段：一个具体的场面（感官层面的，读者能"看到"的画面）
    ↓
第二段：这个场面意味着什么（从具体到判断，但判断必须有事实撑腰）
    ↓
第三段：这个人的矛盾/悖论（所有值得归档的人都有一个内在矛盾）
    ↓
第四段：这个人如何反射了这个时代（不要太大，一个切面即可）
    ↓
结尾：一个具体的落点——这句话、这个画面、这个数字，能让人停一下
```

**长度：350-550 字。** 太短说不透，太长没人读。

## 示例

### 张雪峰

> 2026年3月24日中午12点26分，张雪峰在公司跑步后觉得身体不舒服。下午3点50分，抢救无效。这是他在这个世界上最后的两个小时。
>
> 他生前的最后一场直播说："初中的孩子刚出门，小学的孩子在吃饭，高中的孩子在背书，他们的家长在看我。"——他就是用这种句式，把考研辅导做成了全民娱乐，也把自己做成了一个2656万人关注的符号。
>
> 但很少有人注意他童年真正穷到什么程度。齐齐哈尔富裕县，地名和事实形成反讽。他上大学的钱是全村凑的，睡过火车站，穷得"响都不响"。后来他靠一张嘴实现了阶层跃迁——但也正是这个"怕穷"的底层记忆，让他活成了一条永远在跑的狗。金钱、声望、对手、死亡——他一辈子都在逃跑，从贫穷里逃，从争议里逃，最后还是没跑过心源性猝死。
>
> 他去世后，央视发了讣告。一条没人注意的旧视频被翻出来：有人问他为什么这么拼，他说——"我从小穷怕了。穷过的人，知道每分钱都是命换的。"
>
> 他可能再也不用拼了。

### 峰哥

> 峰哥拍过一条出圈的视频：山东一个精神疾病患者，坐在镜头前用勺子舀空气吃饭。画面安静，尴尬，持续了十分钟。弹幕在笑。但笑完之后评论区沉默了很久——没人说清楚峰哥是在做社会调查还是在消费一个病人。
>
> 这个问题追了他五年。他拍流浪汉、拍癌症青年、拍烂尾楼里的钉子户、拍缅甸诈骗园区里被枪顶着头的人。他的镜头永远对准社会最边缘的角落，而且从来不解释自己为什么要拍。你问他，他就回你一句"这是好事儿啊"——他的第二口头禅。
>
> 2023年9月，他去了一趟缅甸，拍了一集"生死36小时"。他说自己被持枪人员挟持过，差点被当场枪毙，起飞回国时哭了十多分钟。有人扒出细节说不通，断定他是编的。但更多人根本不关心真假——因为在观众心里，他已经不是一个博主了，是一个"亡命天涯"的角色。角色的产出应当符合角色的逻辑。真不真不重要，"好玩"才重要。
>
> 2024年，全平台一起关了他的门。微博、B站、抖音、小红书全面禁止关注。没有明确原因，只知道"违反相关法律法规"。一个总是拍边缘的人，最终把自己也拍到了边缘。
>
> 被封之后他转了股市，2026年拿着手机截图说赚了一个亿。"这是好事儿啊"——这句话从他嘴里说出来，永远有效。你永远无法证明峰哥说的任何一件事是真的还是假的。而这一点，恰恰是他最像这个时代的地方。

### 丁真

> 2020年11月11日，一个摄影师在理塘拍到一个藏族少年来买泡面。随手发到抖音，一夜播放量超过1200万。这就是丁真——他在互联网上的全部原始资本：一张脸，一个眼神，七秒钟视频。
>
> 所有人都在等他开口要价。千万合约、综艺、电影——每一样都在等着他签。但他签了国企。月薪3500元。理塘文旅的总经理说了一句话："我们不希望丁真成为一件商品，我们希望他成为一个人。"
>
> 然后争议来了。"十年寒窗不如丁真一笑"——这是中国互联网男性群体最尖锐的愤怒。一个小学三年级学历的放牛娃，凭什么比我火？凭什么上联合国？凭什么被华春莹推荐？这股愤怒如此之大，以至于"丁真"从一个名字变成了一个测量工具——任何被认为是"德不配位"的人，都会被冠上"XX丁真"的标签。他不是在挨骂，他是在替所有人承受一个社会性焦虑的投射：努力还有没有用？
>
> 但另一组数据很少有人提到：甘孜州旅游收入在丁真走红后的三年里增长了近三倍，理塘游客从15万增长到420万。修路、建学校、修供水工程——因为一个七秒视频，一个高海拔贫困县完成了一次不可逆的经济跃迁。那个放牛娃什么都没做错，他只是站在了这道算术题的两端：一端是私人公正，一端是公共福祉。这两端从来没被同时满足过。
>
> 2026年，他仍然在理塘，仍然骑着马，仍然领着国企的工资。谢霆锋问他为什么那么开心，他说："因为我也很自由。"

## 自检清单

写完读者按后，逐条问自己：

- [ ] 第一段有没有一个具体画面/细节/场景？（没有 → 重写）
- [ ] 我是否说了一句没有事实支撑的抽象判断？（有 → 补具体事实）
- [ ] 换一个人名，这篇还能套上用吗？**（能 → 完全重写）**
- [ ] 我有没有替读者"总结中心思想"？（有 → 删掉，让读者自己感受）
- [ ] 最后一句是一个画面/原话/数字，还是一句空洞的"反映了时代"？（空洞 → 换一个具体的落点）
- [ ] 我自己读到最后一句话，有没有沉默一秒钟？（没有 → 结尾不够好）

## 最后记住

**最好的编者按不是在解释这个人——是在为一个应该被记住的人，留下一个能让人记住的瞬间。**
