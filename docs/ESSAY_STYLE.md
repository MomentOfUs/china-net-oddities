# 档案编者按写作规范

## 定位

「档案编者按」是每个人物页面的核心——它不是百科词条，不是故事会，是**一针见血的人物判词**。

它的功能：用一个短段落，让读者快速理解——这个人为什么值得被归档，他的存在反映了什么社会切面。

## 结构公式

```
一句话定性（犀利判词） + 2-3个关键节点 + 一句社会洞察
```

### 4 句话撑满，不能再多

| 句序 | 功能 | 要求 |
|------|------|------|
| 第1句 | **判词** | 用一句话概括这个人物的本质。不写形容词堆砌，写一个有力的判断。 |
| 第2句 | **节点** | 选一个最能说明问题的经历/事件，带出人物特质。 |
| 第3句 | **延伸** | 从这个事件引出一个更广的观察（社会、时代、群体心理）。 |
| 第4句 | **落点** | 回到这个人，给一个残酷但真实的定论。 |

> 注：如事件极少可压缩为 3 句，最多不超过 5 句。每句控制在 80 字以内。

## 语气要求

| ✅ 要 | ❌ 不要 |
|-------|---------|
| 第一句话就下判断 | 从"XX出生在XX年"开始平铺直叙 |
| 用对比、反讽制造张力 | 堆砌形容词（"他是一个非常……的人"） |
| 保持距离感——编者不站队 | 过度抒情（"他真是太可怜了/太牛了"） |
| 用大白话 | 滥用网络梗当结论 |
| 每句后接具体事实支撑 | 空喊"反映了时代"不举例子 |
| 对一个群体/现象发言时，拿出数据或事实 | 人身攻击（"傻X""脑残"） |

## 开篇句式参考（不要照抄，要自己写）

- "X是中文互联网XX现象最完整的标本——…"
- "X的故事不是一个人的故事，是一个时代的…"
- "要理解X，你得先理解一个道理：…"
- "XX年，一个XX（身份）做了XX（事），然后…"
- "X的荒诞不在于他做了什么，而在于——他什么都没做，却…"
- "如果故事在这里结束，它只是一个…但X没有让故事结束"

## 结尾句式参考

- 不要总结陈词，要留一个能让人停一下的句子
- 反讽式：「那个XX的少年，最终活成了XX」
- 判句式：「XX最大的悲剧/荒诞在于：他以为XX，实际上XX」
- 社会切面式：「X的故事告诉我们——不是XX，而是XX」
- 留白式：用最朴素的一句话收尾，让它自己发酵

## 正例 vs 反例

### 反例（太平淡）
> 张雪峰是考研名师，出生于黑龙江齐齐哈尔。他小时候家里很穷，后来考上了郑州大学。2016年因解读985高校的视频走红。他于2026年3月24日因心源性猝死去世，享年42岁。

### 正例（按规范）
> 张雪峰是最难定义的。——他集老师、网红、商人、老板四种身份于一身，每种都做到了极致。小时候睡火车站，穷得"响都不响"，后来靠一张嘴在互联网上杀出一条路。他去世前最后一次直播说——"初中的孩子刚出门，小学的孩子在吃饭，高中的孩子在背书，他们的家长在看我。"这是他自己写给自己的悼词。

### 反例（太长）
> 阿giao是中国土味网红时代最具代表性的人物之一……（大段人生经历描述，500字）

### 正例（按规范）
> 阿giao的荒诞不在于他有多火——而在于他是一个真正意义上"一无所有"的人，却用一句毫无意义的尾音改变了命运。一个母亲早逝、高中辍学、推着小车卖苹果的农村青年，靠着对着镜头喊"一给我里giao giao"，上了中国新说唱，出了单曲，结了婚，生了孩子。然后又在夜店被人打了一巴掌后，转型成了正能量人生导师。他的"作品"从来不是那首《开心真Giao》——是他自己那个荒诞而完整的人生剧本。

## 示例模板（可直接参考改写）

```
> 判词（一句话定性）
> 这句话的核心是——XX不是XX，而是XX

> 关键节点（选一个最有代表性的场面/事件）
> 用一个具体的场景、一句话、一个数字来支撑判词

> 社会洞察（从个人看到时代）
> 这个人与他所属的群体/时代的关系是什么

> 落点定论（一句道破本质）
> 这个人最终留下了什么/成了什么/说明了什么
```

## 核心原则

1. **短。** 300字以内。每个人都能读完。
2. **狠。** 第一句话就触到本质，不铺垫。
3. **准。** 不说空话，每句话背后都要有具体的事。
4. **冷。** 不煽情，不做道德评判。编者是一把刀，不是一双手。
5. **独。** 每个人的判词都应该是独一无二的——换一个人名就套不上去。

最后一句记住：**不要在结尾告诉读者"应该怎么想"——把事实和判断摆在那儿，读者自己会想。**
