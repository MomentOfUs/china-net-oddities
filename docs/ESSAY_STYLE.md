# 档案编者按写作规范

## 定位

「档案编者按」不是百科词条，不是故事会，是**一篇夹叙夹议的社会学短评**。

它的功能：让读者读完 3-5 分钟后，不只是知道了这个人的经历——而是看到了一个时代切面、一种社会结构性的荒诞、一个值得反思的命题。

每一篇编者按都是一次**微型的学术随笔**——有理论借镜，有社会观察，有具体人物，有人性追问。

## 结构

### 基本框架

```
第一段   判词/定性 —— 这个人是谁？为什么值得被归档？（1-2句）
│
第二段   关键叙事 —— 挑一个最具代表性的节点展开（而非从头讲起）
│
第三段   理论借镜 —— 用一个社会学/哲学概念或框架来分析
│
第四段   社会切面 —— 从个体看到群体/时代/结构
│
第五段   落点定论 —— 回到这个人，留下一个开放的追问
```

### 长度

- **400-600 字**为基准，太浅写不深，太长读不完
- 5-6 个自然段（约 7-10 个 `<p>` 标签）
- 每段 80-120 字左右

## 理论工具箱（请在实际需要时使用，不要生硬嫁接）

编者按需要理论深度，但理论必须**内化**到叙述中，不要突然蹦出一个"根据福柯的规训理论"这种生硬句式。

### 常用框架参考

| 框架 | 核心概念 | 适用场景 |
|------|---------|---------|
| **鲍德里亚 拟像/超真实** | 符号取代真实、拟像的四个阶段 | 造假网红、剧本塌房、人设崩塌——猫一杯、东北雨姐 |
| **戈夫曼 拟剧论** | 前台/后台、印象管理、角色距离 | 人设制造、表演性人格——秀才、嘎子、马保国 |
| **布尔迪厄 文化资本/场域** | 资本兑换、惯习、文化门槛 | 跨圈层走红——丁真、张雪峰、杀马特团长 |
| **福柯 规训/话语** | 话语权力、正常化、监视 | 平台封禁、舆论审判、道德资本——神鹰黑手、药水哥 |
| **德塞托 日常生活实践** | 弱者的战术、消费即生产 | 底层网红的自毁式走红——阿giao、丽丽 |
| **大卫·哈维 时空压缩** | 资本加速、时空重组 | 网红速生速死、梗的生命周期——孙笑川、虎哥 |
| **韩炳哲 功绩社会/倦怠社会** | 自我剥削、积极肯定 | 全民自媒体的灵魂内卷——峰哥、反诈老陈 |
| **桑斯坦 信息茧房** | 选择性暴露、回音室 | 中老年粉丝与秀才、一笑倾城的圈子壁垒 |
| **齐美尔 社会距离** | 陌生人与社会形式 | 丁真的"纯真"被资本定价——距离感本身就是商品 |
| **阿多诺 文化工业** | 标准化、伪个性化 | 土味网红流水线——为什么所有底层网红行为模式如此相似 |
| **勒庞/塔尔德 群体心理 / 模仿律** | 传染、模仿、集体情绪 | 全网寻秦朗、复读"不讲武德" |
| **马克斯·韦伯 祛魅 / 卡里斯马** | 权威类型、魅力领袖 | 张雪峰的卡里斯马型权威是如何建立和崩塌的 |

### 使用技巧

- **不要直接引用理论家名字**。如"戈夫曼会说这是前台表演" → 改为"他有两个人生：镜头前的和镜头后的。前者是精心布置的前台，后者才是他自己。"
- **将理论消化为观察**。如"鲍德里亚的拟像" → "真实的秦朗是否存在已经不重要了——重要的是互联网已经生产了一个足够好的故事，而好故事永远比真相跑得快。"
- **判断标准**：如果去掉理论词条后文章依然成立，说明理论内化成功；如果去除后读者不知道你在说什么，说明理论没消化好。

## 语气

| ✅ 要 | ❌ 不要 |
|-------|---------|
| 第一段就下判断，定调 | 从"XX出生于"开始平铺直叙 |
| 用对比、悖论、反讽制造张力 | 堆砌形容词（"他是一个非常……的人"） |
| 保持距离——编者是一把冷刀 | 过度抒情（"他真是太可怜了"） |
| 具体事实支撑每一句判断 | 空喊"反映了时代"不举例子 |
| 用学术术语时自然融入文本 | 生硬掉书袋（"根据XX理论"）
| 对群体/现象下判断时，有数据/场景支撑 | 人身攻击 |

## 实例：同一人物两种写法

### ❌ 平铺直叙版（太浅）
> 阿giao本名展亚鹏，1991年生于河南许昌。他高中辍学，去广东打过工，后来回老家卖苹果。2017年在快手上因"一给我里giao giao"走红。2018年参加《中国新说唱》被淘汰。2021年在夜店被打。后来转型正能量博主。他的人生很传奇。

### ❌ 4句判词版（太短）
> 阿giao的荒诞在于他竟然靠一句毫无意义的尾音改变了命运。卖苹果的农村青年上了新说唱、结了婚、被打、然后转型成了人生导师。他的"作品"从来不是那首《开心真Giao》——是他自己那套逆袭剧本。

### ✅ 编者按标准版
> 阿giao的故事，拆开来看每一帧都荒诞到令人难以置信，拼在一起却构成了一条完整的底层晋升弧光——它如此完美，以至于它本身就是一个时代命题的注脚。
>
> 一个母亲早逝、高中辍学、推着小车卖苹果的农村青年，花了十年没能改变什么。然后对着镜头喊一句"一给我里giao giao"——命运转过了一个弯。在德塞托的视角下，这是弱者的战术：当正规渠道的门全部关上的时候，一无所有的人只能选择"自我献祭"来获取流量——把自己变成一个可供消费的符号，然后等待互联网的分配。
>
> 他成功了。上了《中国新说唱》，出了单曲，结了婚，生了孩子。然后在夜店被人从后脑拍了一巴掌——全网围观。再然后，他撕掉"土味"的标签，穿上"正能量"的外衣，站在镜头前讲人生道理。这个转型不是道德觉醒，而是一种场域间的资本兑换：在土味场域积累的注意力，换一个场域重铸为励志话语。布尔迪厄如果活着，会点头。
>
> 但拆解他需要的不只是理论。更深的问题是：一个除了"一无所有"之外什么都没有的人，凭什么在互联网上获得一切？答案残酷而直接——因为互联网需要故事，而他的故事恰好符合底层逆袭的经典叙事。平台需要他，因为他是"普通人也能改变命运"的活广告。观众需要他，因为他证明了那一丝微茫的可能性——即使是我，也可能有一天"giao"一声飞起来。
>
> 到2025-2026年，giao哥还在更新。如果说有什么比"一给我里giao giao"更值得记住的，那就是这个事实本身：一个被互联网选中的人，正在努力证明自己配得上这份运气。他可能永远不会成功——但谁在乎呢？互联网早就忘了他为什么出名，只记得他还在。

### ✅ 理论内化示例：猫一杯
> 猫一杯的故事不是关于造假的，是关于**叙事的生产速度超过了事实的验证速度**。
>
> 一个千万粉博主，在巴黎街头"捡到"了一本中国小学生的寒假作业。她对着镜头播报，全网配合演出了一场寻人运动。这个视频的病毒式传播不是偶然——它精准踩中了每一个传播学上的引爆点：异国元素、文化认同、情绪价值、完美闭环。秦朗是否存在不重要——重要的是互联网已经生产了一个好故事，而好故事永远比真相跑得快。
>
> 这是鲍德里亚预言的"超真实"已经完全实现的世界：一个自导自演的剧本，被全民当作真实事件消费；当警方通报真相时，人们愤怒的不是造假本身，而是"这个好故事居然是假的"。消费的从来都不是真相，消费的永远是叙事。
>
> 猫一杯被封号了。但她留下了一个教科书级的案例：注意力经济的底层逻辑不是"内容为王"，是"故事为王"。而当故事被证明是假的，愤怒的观众会转头吃掉讲故事的人。这是每个网红都懂的恐惧，也是每个网红都假装不存在的终局。

## 核心原则

1. **有理论含量，但不掉书袋。** 消化后说出来，不贴标签。
2. **有社会切面，但不宏大叙事。** 从一个具体的人，看一个时代的褶皱。
3. **有真观察，但不自以为是。** 留一个开放的问题给读者，不做全知全能的神。
4. **有锐度，但不是骂人。** 编者是一把解剖刀，不是一把杀猪刀。
5. **每个人物都是唯一的。** 换一个人名，这篇文章就应该完全不成立。

### 定稿前自问

- 去掉这个人名，这篇文章还能套到另一个人头上吗？→ 能 => 重写
- 我有没有用到至少一个理论框架来组织判断？→ 无 => 太浅
- 最后一段让读者想了一会儿，还是直接滑走了？→ 滑走 => 结尾太弱
- 我有没有说了一句自己也不确定对不对的大话？→ 有 => 删掉
