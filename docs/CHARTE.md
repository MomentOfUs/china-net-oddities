# 入选标准 · 评价体系

## 总纲

> 这里没有网红，只有在围观与挣扎中，被时代重力碾成的命运标本。

**"互联网奇观"不等同于"网红"。**
奇观是时代的切片——一个人被卷入或主动制造的文化现象，其存在本身就揭示了互联网时代的某种本质。

---

## 壹 · 核心定义

**互联网奇观**是指在互联网上形成的、同时具备以下全部特征的公众人物或群体现象：

| 标准 | 含义 |
|------|------|
| **现象级** | 跨越圈层，形成广泛社会讨论 |
| **不可复制** | 其走红路径无法被刻意模仿 |
| **时代标本性** | 承载着某个时期的社会情绪或文化矛盾 |
| **叙事完整性** | 有起承转合，构成一个可供讲述的故事 |

> 一个合格的奇观人物，其存在本身就值得被叙述——不是因为流量，而是因为象征意义。

---

## 贰 · 六维评价体系

每个候选人物按六个维度评分，满分 **60** 分。准入线 **36** 分。

### 一 · 国民度（0-10）

大众知晓程度，不限于核心受众圈。

| 分数 | 含义 |
|------|------|
| 0-3 | 小圈层认知 |
| 4-6 | 同类群体内知名 |
| 7-8 | 跨越多个圈层 |
| 9-10 | 全民认知 |

参照：丁真 9 · 吴京 10

### 二 · 争议性（0-10）

讨论烈度与持续时长。零争议不构成奇观。

| 分数 | 含义 |
|------|------|
| 0-2 | 几乎无争议 |
| 3-5 | 有讨论但热度短暂 |
| 6-8 | 长期争议，形成对立阵营 |
| 9-10 | 社会级撕裂 |

参照：三和大神 7 · 范小勤 8

### 三 · 荒诞性（0-10）

离奇、意外、反常识。荒诞是奇观的核心美学。

| 分数 | 含义 |
|------|------|
| 0-2 | 普通、可预期 |
| 3-5 | 有些意外但合理 |
| 6-8 | 明显反常识 |
| 9-10 | 超出想象 |

参照：王伟恒 8 · 良子 8

### 四 · 时代切片性（0-10）

能否作为某个时代节点的文化锚点。

| 分数 | 含义 |
|------|------|
| 0-2 | 无关时代背景 |
| 3-5 | 可以对应某个现象 |
| 6-8 | 时代情绪的具体载体 |
| 9-10 | 成为时代的注脚 |

参照：三和大神 10（注脚一个时代）

### 五 · 叙事弧光（0-10）

故事的起承转合，是否有"剧本感"。

| 分数 | 含义 |
|------|------|
| 0-2 | 平淡无起伏 |
| 3-5 | 有小转折 |
| 6-8 | 明确的起承转合 |
| 9-10 | 完整戏剧弧光 |

参照：张雪峰 9 · 范小勤 9

### 六 · 互联网原生性（0-10）

这个人的名气和现象靠互联网到什么程度。

| 分数 | 含义 |
|------|------|
| 0-2 | 线下知名度为主 |
| 3-5 | 线上线下各半 |
| 6-8 | 互联网是主要舞台 |
| 9-10 | 完全因互联网而生 |

参照：丁真 9（7秒撑起全部）· 吴京 1

---

## 叁 · 否决条件

具备以下任意一项，即使六维评分达标也不收录：

| 否决项 | 说明 |
|--------|------|
| **纯营销产物** | MCN按公式批量生产的达人，缺乏不可复制性 |
| **纯猎奇消费** | 单纯消费身体缺陷或苦难，不承载文化意义 |
| **信息不确** | 核心事实模糊，无法构建可信叙事 |
| **运营者本人** | 不收录自己人 |
